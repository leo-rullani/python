Hinweise zu Python

1. Variablen

  • Variablen speichern Werte. Zuweisung erfolgt mit =.
  • Beispiel: name = "Anna" oder age = 25.
  • Python erkennt Datentypen automatisch (String, Zahl, Boolean etc.).
  • Mehrere Zuweisungen möglich: name, age = "Lisa", 30.
  • Werte tauschen: name, age = age, name.
  • Für Ausgaben: print(name) oder mit f-String: print(f"Hallo {name}").
  • Variablennamen dürfen nicht mit einer Zahl beginnen und keine Sonderzeichen enthalten.
  • Keine Keywords wie if, class, def als Namen verwenden.
  • Groß- und Kleinschreibung wird unterschieden: name ≠ Name.
  • Vermeide „Magic Numbers“ – nutze stattdessen benannte Variablen.
